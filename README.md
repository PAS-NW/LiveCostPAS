# PAS Groundworks Prototype v0.1

PAS Groundworks is a modular commercial management platform for PAS NW Construction Ltd.

This prototype focuses on the core platform shell and Live Costs module.

## Includes
- Premium Streamlit UI shell
- Left module navigation
- Executive dashboard
- Live Costs module layout
- History and Compare screens
- Settings placeholders
- Database-ready structure
- SharePoint-ready service placeholders

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
